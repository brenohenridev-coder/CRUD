CRUD de Cadastro de Usuários                      

Projeto CRUD desenvolvido em Python com o objetivo de realizar o cadastro, leitura, atualização e exclusão de usuários,
armazenando informações pessoais de forma organizada.

---

Funcionalidades

- Cadastrar usuários
- Listar usuários cadastrados
- Atualizar dados de um usuário
- Remover usuários
- Buscar usuário por CPF ou nome

---

Dados armazenados

Cada usuário possui os seguintes dados:

- Nome
- Idade
- E-mail
- CPF
- Celular
- Data de nascimento

---

Tecnologias utilizadas

- Python 3.x  
- Banco de dados (SQLite / MySQL / arquivo local)
